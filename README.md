# Dev script:

Run the project localy: 
`expo start`

# Build scripts:

Build for android:
`eas build --platform android`

Deploy on Google Play Console:
`eas submit --platform android`

Build and deploy on Google Play Console: 
`eas build --platform android --auto-submit`
